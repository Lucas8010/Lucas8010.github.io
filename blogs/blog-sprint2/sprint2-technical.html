IDs

This is one of the important attributes when learning css, these are able to divide content allowing for easy content controlling for yourself and your team. IDs are unique, This means that each element can only have one ID and not multiples. You are able to see that I have used the ID Attribute to only to identify the each boxes so that I can use css to easily edit them to my liking, as shown in the second picture. I would only use the ID attribute whenever I tried creating a section/part of my website like the grid section.

Classes are the basically almost the same as Id's but you are able to apply multiple strings/classes to the same element. Making it not Unique. Using classes means that you are able to add multiple strings to one class instead of making a whole bunch of seperate classes in the same element.
